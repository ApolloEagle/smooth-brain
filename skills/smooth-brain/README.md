# smooth-brain skill

Reduces AI response verbosity and complexity. Three presets:

| Preset | Effect |
|--------|--------|
| `wrinkled` | Plain English, no jargon, no filler openers |
| `bumpy` | Short sentences, common words, bullets over prose *(default)* |
| `smooth` | Explain-like-I'm-5, one sentence answers, analogies only |

## Usage

```
/smooth-brain              # bumpy (default)
/smooth-brain wrinkled
/smooth-brain bumpy
/smooth-brain smooth
```
